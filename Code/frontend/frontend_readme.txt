本文档提供对于前端操作相应的指示。
前端需求nodejs以启动。
nodejs的下载网址为：https://nodejs.org/zh-cn
下载后正常安装即可使用npm指令。

第二步是安装vite和axios，前置指令（定位）为cd code\frontend\sosfrontend。
安装指令为：
npm install vite
npm install axios
（需要梯子）

前端启动的第0步为启动后端，需要有后端作为支持才可以使用前端的各项功能。
后端启动指令见backend/main_frame内注释。

前端启动指令为npm run dev。
启动后进入 http://localhost:5173/ 即可访问。
终端会展示帮助指令，退出前端在终端输入q+enter即可。

当前前端仅包含长文本输入功能，且目前传输有问题，仅供测试。
